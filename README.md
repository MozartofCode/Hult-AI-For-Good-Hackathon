   ```sh

   ```
2. Install dependencies:  
   ```sh
   npm install  # For Node.js

   ```
3. Start the Flask backend server:  
   ```sh
   flask run
   ```
4. Start the Node.js backend (if needed):  
   ```sh

   ```
